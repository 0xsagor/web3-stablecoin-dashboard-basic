# Web3 Stablecoin Dashboard Basic

This project helps beginners learn how stablecoins work and how to track them in a Web3 app.

You will learn:
- What stablecoins are
- How to connect a wallet
- How to read token balances
- How to calculate total value
- How crypto dashboards work

## Requirements
- Node.js installed
- MetaMask browser extension
- Basic JavaScript knowledge

## Setup

1. Clone the repo
git clone https://github.com/yourusername/web3-stablecoin-dashboard-basic.git

2. Install packages
npm install

3. Run the app
npm start

4. Open browser
http://localhost:3000

## Project Structure

- index.html = Main webpage  
- app.js = Frontend logic  
- tokens.js = Token balance functions  
- web3.js = Blockchain connection  
- api.js = Price API handler  
- config.js = Network settings  
- server.js = Local server  
- package.json = Project info  
- utils.js = Helper functions  
- .env.example = Environment example  

## Features
- Connect wallet  
- Show USDT balance  
- Show USDC balance  
- Show total value in USD  
- Simple dashboard  

## Learning Goal
This project explains how stablecoins can be tracked in a Web3 app.
